Curtains ERP

Complete User Manual — Staff, Accounts & Administration

Version 1.0 April 2026 Elite Curtains Ghana

IntroductionSystem Overview

This manual covers every feature of the Curtains ERP system. Part 1 is written for everyday staff (Sales, Workshop). Part 2 covers Accounts workflows. Part 3 covers system administration for Admins.

Curtains ERP is a web-based business management system built for curtain manufacturing and retail businesses. It covers the full commercial cycle — from customer enquiry and quoting through to production, delivery, invoicing, and financial reporting.

The system runs entirely in a web browser. No installation is required on individual computers. All data is stored centrally on the company server and accessible to authorised staff from any device on the network.

IntroductionUser Roles

Every user account is assigned one of four roles. The role controls which pages and actions are available to that user.

RoleTypical PersonAccess Level
ADMIN Business owner / Manager Full access to everything including settings, user management, BOM templates, and all approval actions.
ACCOUNTS Accountant / Finance officer Full financial access — invoices, purchasing, expenses, reports, inventory. Can action approvals. Cannot manage users or change system settings.
SALES Sales representative Customers, quotes, orders. Can view the dashboard and production status. Cannot access financial reports or purchasing.
WORKSHOP Production staff / Tailor Production job cards, inventory (view and issue materials). Can see orders. No financial access.
ℹ️
If you cannot see a menu item or a button, it is likely because your role does not include that permission. Contact your system administrator if you believe you need additional access.

IntroductionLogging In

Open your web browser and navigate to the system URL provided by your administrator (typically http://localhost:5173 on the local network).

  1. Enter your email address in the Email field.

  2. Enter your password in the Password field.

  3. Click Sign In. You will be taken to the Dashboard.

📸 Screenshot: Login Page
Capture the login screen showing the email/password form and the violet "Sign In" button.

First-Time Login / Password Change

If your account was newly created by an administrator, you will be redirected to a Change Password screen immediately after your first login. You must set a new password before you can access the system. Passwords must be at least 8 characters.

⚠️
Never share your password. Each staff member must have their own account. Shared accounts make it impossible to trace who performed which action.

Chapter 2 · Staff GuideCustomers

The Customers module stores all client records including contact details and window measurements. Accurate customer records are essential for quotes and invoices.
📸 Screenshot: Customers List
Capture the Customers page showing the table of customers with name, phone, email, and total spent columns.

Viewing Customers

Navigate to Customers in the sidebar. The page shows a searchable, paginated table of all customers. Use the search bar at the top to find a customer by name, phone, or email.

Adding a New Customer

  1. Click the New Customer button (top right of the page).

  2. Fill in the customer's Name (required), phone, email, and address.

  3. Click Save. The customer is added to the list immediately.

💡
At minimum, always record the customer's name and phone number. An email address enables the system to send quotes directly to the customer.

Recording Window Measurements

Window measurements are stored against each customer and can be pre-loaded when creating a quote. This saves time for repeat customers.

  1. Open the customer record by clicking their name in the list.

  2. Scroll to the Windows section and click Add Window.

  3. Enter the Room Name (e.g. "Living Room"), Width (cm), Drop (cm), and Quantity.

  4. Click Save. Repeat for each window or room.

FieldDescription
Room NameLabel to identify the window location. Used on quotes and job cards.
Width (cm)The track or rod width in centimetres.
Drop (cm)The finished drop length from track to hem in centimetres.
QuantityNumber of identical windows in this room (e.g. 2 matching windows).

Chapter 3 · Staff GuideQuotes

Quotes are price proposals generated for customers. The system automatically calculates material costs using BOM templates, so quotes are fast and consistent.
📸 Screenshot: Quotes List
Capture the Quotes page showing the list of quotes with status badges, customer names, and totals.

Quote Statuses

StatusMeaning
DRAFTBeing prepared, not yet sent to the customer.
SENTPresented or emailed to the customer, awaiting their response.
ACCEPTEDCustomer has accepted. Can be converted to an order.
REJECTEDCustomer declined the quote.

Creating a New Quote

  1. Click New Quote on the Quotes page.

  2. Select the Customer from the dropdown. If the customer is new, go to the Customers module first and create their record.

  3. Set a Valid Until date (e.g. 30 days from today). After this date the quote is marked expired.

  4. Add line items — one for each window or group of windows. Click Add Item for each line.

  5. For each line item, complete the fields described in the table below.

  6. Review the cost summary at the bottom (material cost, labour, overhead, markup, total).

  7. Click Create Quote. The quote is saved as DRAFT.

📸 Screenshot: New Quote Form
Capture the quote creation form showing customer selection, line items editor with curtain type/BOM/fabric fields, and the cost summary at the bottom.

Quote Line Item Fields

FieldDescription
Window LabelName for this window (e.g. "Master Bedroom" or "Living Room Left"). Pulled from customer windows if set.
Curtain TypeThe style: Eyelet, Pinch Pleat, Roman Blind, etc.
BOM TemplateThe recipe/formula to use. Auto-populates when curtain type is selected.
Width (cm)Track/rod width. Auto-fills from customer windows.

💡
When you change the Fabric Material, the system recalculates the material cost using that fabric's unit cost and the formula in the BOM template. The suggested selling price updates automatically.

Applying a Discount

A percentage discount can be applied to the entire quote total.

  1. In the quote form, find the Discount % field below the line items table.

  2. Enter the discount percentage (e.g. 5 for 5%).

  3. The subtotal, discount amount, VAT, and total fields update immediately.

⚠️
Discount Approval Gate: If the discount exceeds the configured threshold (default: 10%), the quote is saved with an Awaiting Approval status. It cannot be converted to an order until an Admin or Accounts user approves it in the Approvals module. See Chapter 8.

Converting a Quote to an Order

Once a customer accepts a quote, convert it to a confirmed order.

  1. Open the quote by clicking its number in the Quotes list.

  2. Click Mark as Accepted (changes the quote status to ACCEPTED).

  3. Click Convert to Order.

  4. Enter the Deposit Amount (GHS) received from the customer.

  5. Set the Delivery Date if known.

  6. Click Confirm. An order is created and a job card is automatically generated for the workshop.

⚠️
High-Value Order Approval: If the order total exceeds the configured threshold (default: GHS 5,000), the order is created in a PENDING state pending management approval. Production cannot begin until it is approved. See Chapter 8.
ℹ️
A quote with a pending discount approval cannot be converted to an order. The Convert button will be disabled until the discount is approved or reduced below the threshold.

Chapter 5 · Staff GuideProduction

The Production module is primarily used by Workshop staff. It shows job cards for each order, tracks what materials need to be issued, and lets staff update the progress of each job.
📸 Screenshot: Production Page
Capture the Production page showing the list of job cards with order numbers, customer names, status badges, and material issuance status.

Understanding Job Cards

A job card is automatically created when an order is confirmed. One job card covers all the curtains in one order. The job card lists every material required (fabric, lining, heading tape, eyelets, weights, etc.) calculated from the BOM template, and tracks which materials have been physically issued from the store.

FieldDescription
Job NumberUnique identifier e.g. JOB-2026-0042
Order NumberThe order this job belongs to
CustomerCustomer name for reference
StatusPENDING → IN_PROGRESS → COMPLETED
Assigned ToWorkshop staff member responsible
MaterialsList of required materials with required and issued quantities

Issuing Materials

Before cutting can begin, materials must be issued from the store.

  1. Open the job card by clicking the Job Number.

  2. Review the Materials Required list. Each line shows the material code, description, required quantity, and issued quantity.

  3. Click Issue Materials next to each line once the physical material has been taken from stock.

  4. The stock level for that material is automatically reduced.

  5. Once all materials are issued, click Start Job to move the job to IN_PROGRESS.

⚠️
Only issue materials when you are physically taking them from the store. Issuing without taking the material causes stock discrepancies.

Updating Production Status

Workshop staff update the status of each order line item as work progresses:

  • PENDING → Not yet started
  • CUTTING → Fabric is being cut
  • SEWING → Stitching in progress
  • COMPLETED → Finished and quality-checked

When all items in a job are COMPLETED, the job card status is automatically set to COMPLETED and the linked order moves to the COMPLETED stage.

Chapter 6 · Staff GuideInvoices & Payments

Invoices are the financial documents sent to customers for payment. They are created from completed orders and track the full payment history.
📸 Screenshot: Invoices List
Capture the Invoices page showing the table with invoice numbers, customer names, issue dates, totals, amounts paid, balances, and status badges.

Invoice Statuses

StatusMeaning
DRAFTCreated but not yet sent to customer.
SENTEmailed or handed to customer.
PARTIALSome payment received but balance remains outstanding.
PAIDFully paid. Balance is GHS 0.
OVERDUEDue date has passed with outstanding balance. Set automatically by the system.
CANCELLEDInvoice has been voided. Requires approval if it was SENT or PAID.

Creating an Invoice

ℹ️
Who can generate invoices: Only Admin and Accounts users can create invoices. Sales and Workshop users will see a note on completed orders directing them to contact Accounts.

Invoices are generated directly from the Orders page — not from the Invoices page. The Invoices page is for viewing, payment recording, and emailing only.

  1. Navigate to Orders and open a COMPLETED or DELIVERED order by clicking View.

  2. Scroll to the Invoice section at the bottom of the order detail. Click Generate Invoice.

  3. The system automatically:
    • Pulls all line items from the order
    • Applies the current VAT rate from Settings
    • Credits any deposit already recorded on the order
    • Sets the due date to 30 days from today
    • Marks the order status as DELIVERED

  4. The invoice is created as DRAFT. The order detail now shows the invoice number instead of the Generate button.

  5. Go to Invoices, find the new invoice, and click Mark as Sent when ready to share with the customer — or click Email to Customer to send it directly by email (requires an email address on the customer record).

ℹ️
The invoice total includes the deposit already paid. The system shows both the total and the balance due (total minus any payments recorded). You cannot adjust line items on the invoice — if the amounts are wrong, correct the order first before generating.

Recording Payments

  1. Open the invoice by clicking its number.

  2. Click Record Payment.

  3. Enter the Amount (GHS), Payment Method, Payment Date, and an optional Reference (e.g. mobile money transaction ID).

  4. Click Save Payment. The invoice balance updates immediately.

  5. When the balance reaches GHS 0, the invoice is automatically marked as PAID.

Capture the "Record Payment" modal showing amount, method, date, and reference fields.

Accepted Payment Methods

  • Cash
  • Bank Transfer
  • Mobile Money (MTN, Telecel, AirtelTigo)
  • Cheque

Requesting Invoice Cancellation

To cancel an invoice that has already been sent or paid:

  1. Open the invoice.

  2. Change the status to Cancelled using the status dropdown.

  3. The system will not cancel it immediately. Instead, a cancellation request is submitted to the Approvals queue.

  4. An Admin or Accounts manager will review and approve or reject the cancellation. You will see the Awaiting Approval badge on the invoice until a decision is made.

🚫
Cancelling a PAID invoice is a significant financial action. It will only be approved if there is a valid business reason. Provide a clear explanation when making the request.

  2. Enter the Date, Amount (GHS), and Description (e.g. "Electricity bill – April").

  3. Select the Type: Admin, Factory, or Shared (see below).

Expense Types Explained

TypeExamplesHow it's used

Filtering Expenses

Use the From / To date pickers and the Type dropdown to filter the expense list to a specific period or expense type. The total for the filtered view is shown in the filter bar.

  3. Click Save.

Manual Stock Adjustments

Use manual adjustments to correct discrepancies found during a stocktake, or to record damaged/destroyed stock.

  1. Open the material record.

  2. Click Adjust Stock.

  3. Select the Movement Type:

    • RETURN — goods returned to stock (immediate, no approval needed)
    • MANUAL_ADJUSTMENT — stocktake correction (requires approval)
    • DAMAGE — write-off of damaged or destroyed stock (requires approval)

  4. Enter the Quantity (positive to add, negative to remove) and a Note explaining the reason.

  5. Click Submit.

⚠️
MANUAL_ADJUSTMENT and DAMAGE movements require approval. The stock level does not change until an Admin or Accounts manager approves the request in the Approvals module. You will see a PENDING badge on the movement until then.

Stock Movement History

Every change to a material's stock is recorded with a timestamp, movement type, quantity, and the user who made the change. Open any material record and scroll to the Movements section to see the full audit trail.

Low Stock Alerts

When a material's current stock falls to or below its Minimum Stock level, it appears in the low-stock alert card on the Dashboard and is highlighted in the Inventory list. Raise a purchase order promptly to avoid production delays.

Chapter 10 · Accounts GuideBOM Templates

BOM (Bill of Materials) Templates define the material recipes for each curtain type. When a quote line item is created, the system uses the BOM template to calculate exactly how much of each material is needed and what it costs.
ℹ️
BOM templates are typically set up once by the Admin and rarely need changing. Modifying a template does not affect existing quotes or orders — only new quotes use the updated template.

Curtain Types vs BOM Templates

A Curtain Type is the style name (e.g. Eyelet, Pinch Pleat, Roman Blind). A BOM Template is the specific recipe attached to a curtain type — one curtain type can have multiple templates (e.g. a standard and a premium version).

Creating a BOM Template

  1. Navigate to BOM Templates and click New Template.

  2. Select the Curtain Type and give the template a descriptive Name.

  3. Set the Default Fullness Ratio (e.g. 2.0 for eyelet, 2.5 for pinch pleat).

  4. Enter Labour Hours — the standard time in hours to make one unit of this style. Used to calculate labour cost.

  5. Optionally set a Labour Hours Formula if the time varies by window size (advanced — leave blank to use the fixed hours).

  6. Add Line Items — one for each material in the recipe.

  7. Click Save Template.

📸 Screenshot: BOM Template Editor
Capture the BOM template form showing the header fields (curtain type, name, fullness, labour hours) and the line items table with material, formula, role, and sort order columns.

BOM Line Item Fields

FieldDescription
MaterialThe default material for this line. Can be overridden per quote.
Quantity FormulaA JavaScript expression that calculates the required quantity. Available variables: width_m, drop_m, fullness_ratio, fabric_width_m.
RoleFIXED, FABRIC, or LINING. See below.
Sort OrderDisplay order on job cards and quote breakdowns.

Fabric and Lining Roles

The Role field on a BOM line item controls whether that line's material can be substituted when creating a quote:

RoleBehaviour
FIXEDThe material is always this specific item — it cannot be changed on a quote. Use for heading tape, eyelets, hooks, weights.
FABRICThis line's material is the main fabric. When a quote specifies a different fabric, this line's cost recalculates using the selected fabric's price and the same quantity formula.
LININGThis line's material is the lining fabric. Substituted with the lining selected on the quote.
💡
Every template should have exactly one FABRIC line and (if applicable) one LINING line. Without these roles set correctly, fabric substitution on quotes will not work and costs will not recalculate.

Formula Examples

FormulaWhat it calculates
(width_m * fullness_ratio) / fabric_width_mMetres of fabric needed for the panel width
width_m * fullness_ratio + 0.2Metres of heading tape (width × fullness + 20cm for joins)
Math.ceil(width_m * 4) + 2Number of eyelet rings (4 per metre + 2 end rings)
(width_m + 0.1) * (drop_m + 0.3)Square metres of fabric for a roman blind with seam allowance
1Fixed quantity of 1 regardless of dimensions (e.g. 1 pack of fixings)

Chapter 11 · Accounts GuidePurchasing

The Purchasing module manages the full procurement cycle — from supplier records through purchase orders, goods receipt, and landed cost allocation.

Suppliers

Supplier records store contact details and preferred currency. Open the Suppliers tab on the Purchasing page to view and manage them.

Adding a Supplier

  1. Click New Supplier.

  2. Enter the Company Name (required), Contact Person, Phone, Email, Address, and Preferred Currency.

  3. Click Save.

💡
Adding the supplier's email address enables the system to send purchase orders directly from the Purchasing page. Without an email, you must download the PDF and send it manually.

Purchase Orders

A Purchase Order (PO) is a formal document sent to a supplier requesting specific materials at agreed prices.

Creating a Purchase Order

  1. On the Purchase Orders tab, click New PO.

  2. Select the Supplier.

  3. Set an Expected Date (when you expect delivery).

  4. Add line items: for each material, select the material, enter the Qty Ordered and Unit Cost (USD).

  5. Optionally enter landed cost estimates: Freight (USD), Port Clearing (GHS), Other (GHS). These are estimates at this stage and can be adjusted when the goods arrive.

  6. Click Create PO. The PO is saved as DRAFT.

Submitting a PO for Approval

PO Approval Workflow
Create PO → saved as DRAFT
↓
Click Submit for Approval → PO flagged Awaiting Approval
↓
Admin/Accounts approves → PO flagged Approved
↓
Click Send to Supplier → PO emailed; status becomes SENT

  1. Open the PO by clicking its number.

  2. Click Submit for Approval. The button disappears and an "Awaiting Approval" badge appears.

  3. Once approved, the Send to Supplier button appears. Click it to email the PO PDF directly to the supplier.

  4. If the supplier has no email on file, download the PDF and send it manually, then click Mark as Sent.

ℹ️
A PO cannot be sent to a supplier until it has been approved. This ensures all orders are reviewed before commitments are made.

Goods Receipt Notes (GRNs)

When goods arrive, a Goods Receipt Note (GRN) records exactly what was received and updates stock levels accordingly.

  1. Open the PO that was delivered.

  2. Click Record Goods Receipt.

  3. For each line item, enter the Qty Received (this may be less than ordered for partial deliveries) and confirm or adjust the Unit Cost (USD).

  4. Enter the actual landed costs from the supplier invoices: Freight, Port Clearing, and Other charges.

  5. Click Confirm Receipt. Stock levels are updated immediately using Weighted Average Cost (WAC).

💡
Partial Receipts: If only part of the order arrives, enter only the quantity actually received. The PO status will change to PARTIALLY_RECEIVED and remain open for the remainder. You can record another GRN when the rest arrives.
ℹ️
Weighted Average Cost (WAC): Each time goods are received, the material's unit cost is recalculated as a weighted average of existing stock cost and new receipt cost. This ensures the cost of materials issued to production reflects the true average purchase price.

Landed Cost Entries

Sometimes freight and clearing invoices arrive days after the goods. The Landed Costs tab allows you to allocate these costs across multiple GRNs after the fact, updating material costs without re-entering a GRN.

Creating a Landed Cost Entry

  1. Open the Landed Costs tab and click New Landed Cost Entry.

  2. Enter the cost amounts from your freight and clearing invoices.

  3. Select the GRNs that were part of this shipment. The system shows the available GRNs that have not yet been assigned a posted landed cost entry.

  4. Review the allocation preview — costs are distributed proportionally based on each GRN's value in USD.

  5. Click Save as Draft. Review the allocation, then click Post to finalise.

⚠️
Once a Landed Cost Entry is Posted, it cannot be edited. The material unit costs are updated permanently. Double-check all amounts before posting.

Chapter 12 · Accounts GuideReports

The Reports module provides financial summaries for management decision-making, tax compliance, and cost analysis.

Sales Report

Shows revenue for a selected date range, broken down by customer and order. Includes:

  • Total revenue (from paid and partially paid invoices)
  • Discounts given
  • Revenue by customer
  • Outstanding balances

Set the From and To dates using the date pickers at the top of the report, then click Generate.

Overhead Summary Report

Shows the total operating costs for a selected period and calculates the suggested overhead rate per labour hour:

  • Factory overhead total (direct factory + 65% of shared)
  • Admin overhead total (direct admin + 35% of shared)
  • Suggested overhead rate (GHS per labour hour) based on your configured capacity hours
💡
Use the Overhead Summary report monthly to review whether your Overhead Rate setting in Settings still accurately reflects actual costs. If costs have risen significantly, update the rate so future quotes absorb the correct overhead.

VAT Report

Summarises VAT collected on invoices for a selected period. Used to prepare VAT returns. Shows:

  • Total net sales (ex-VAT)
  • Total VAT collected
  • Total gross invoiced (inc-VAT)
  • Breakdown by invoice
ℹ️
VAT is only included on invoices where the VAT Rate setting is greater than 0. Set the VAT rate in Settings → General → VAT Rate (e.g. 0.15 for 15%).

Business Information

Found in Settings → General → Business Information. These values appear on printed quotes, purchase orders, and invoices.

SettingDescription
Business NameYour company name as it appears on all documents.
PhoneCompany contact phone number.
EmailCompany email address for correspondence.
AddressPhysical address printed on invoices and POs.
Default Markup RatioThe margin added on top of total cost to suggest a selling price. E.g. 0.35 = 35% markup.

Exchange Rate Management

Found in Settings → Currency. The exchange rate (GHS per 1 USD) is used to convert material costs and produce GHS-equivalent prices on quotes and invoices.

  1. The current rate and the date it was last set are shown at the top of the Currency tab.

  2. To update, enter the new rate in the New Rate field (e.g. 16.2500).

  3. Click Update Rate. The new rate is saved and all material GHS costs recalculate.

⚠️
Updating the exchange rate recalculates the GHS cost of all materials immediately. Existing quotes and invoices are not affected — they store a snapshot of the rate at the time they were created. Only new quotes will use the updated rate.
💡
Admin users can also update the exchange rate from the sidebar shortcut (the pencil icon next to the rate widget at the bottom of the sidebar) without navigating to Settings.

Production Cost Rates

Found in Settings → General → Production Costs. These rates drive the labour and overhead component of BOM-based quote pricing.

SettingDescription
Labour Rate (GHS/hour)How much one hour of tailoring labour costs. Multiplied by the BOM template's Labour Hours to get labour cost per curtain.
Overhead Rate (GHS/hour)Factory overhead apportioned per labour hour. Calculate as: total monthly factory overhead ÷ total productive hours per month. E.g. GHS 2,000 overhead ÷ 200 hours = GHS 10/hour.

Example: A roman blind BOM says 2.5 labour hours. With Labour Rate = GHS 15/hr and Overhead Rate = GHS 10/hr, the labour+overhead cost per blind = (2.5 × 15) + (2.5 × 10) = GHS 37.50 + GHS 25 = GHS 62.50.

Approval Thresholds

Found in Settings → General → Approval Thresholds. These three values control when the approval gate is triggered for expenses, discounts, and high-value orders.

SettingDefaultDescription
Expense Approval Threshold (GHS)500Expenses above this amount require approval before being included in reports.
Quote Discount Threshold (%)10Quote discounts above this percentage require approval before the quote can be converted to an order.
Order Total Threshold (GHS)5,000Orders converted from quotes with a total above this amount require approval before confirmation.
💡
Set these thresholds based on your business's tolerance for risk. For tighter control, lower the thresholds. For a small team with high trust, raise them to reduce approval friction.

After making any changes in the General tab, click Save Settings at the bottom of the page.

  3. Click Create User.

  4. Give the staff member their email and temporary password. On their first login, they will be prompted to set a new password.

Choose the role carefully — it determines everything the user can see and do. See the Roles table in the Introduction section and the full Permissions Matrix in Appendix A.

Deactivating a User

When a staff member leaves, deactivate their account to prevent access. You cannot delete users (to preserve audit history), but a deactivated account cannot log in.

  1. In the Users table, find the user and click the shield-off icon on the right.

  2. Confirm the deactivation. The user's status changes to Inactive.

🚫
You cannot deactivate your own account. If the admin account owner is leaving, have another admin deactivate it.

Appendix BApproval Flows Reference

This reference shows exactly what triggers each approval type and what happens at each outcome.

1. Purchase Order Approval

Trigger

Every DRAFT PO — Accounts/Admin clicks "Submit for Approval"

On Approve

PO approvalStatus → APPROVED. "Send to Supplier" button becomes available.

On Reject

PO approvalStatus → REJECTED. Requester must edit the PO and re-submit.

2. Expense Approval

Trigger

Expense amount > approval.expenseThresholdGhs (default GHS 500)

On Approve

Expense approvalStatus → APPROVED. Included in overhead reports.

On Reject

Expense approvalStatus → REJECTED. Excluded from reports.

3. Quote Discount Approval

Trigger

Quote discount rate > approval.quoteDiscountThresholdPct (default 10%)

On Approve

Quote approvalStatus → APPROVED. "Convert to Order" becomes available.

On Reject

Quote approvalStatus → REJECTED. Sales must reduce the discount and re-save the quote (which re-triggers the approval if still above threshold).

4. High-Value Order Conversion

Trigger

Order total > approval.orderTotalThresholdGhs (default GHS 5,000)

On Approve

Order status → CONFIRMED. Job card activated for production.

On Reject

Order stays PENDING. No production begins.

5. Invoice Cancellation

Trigger

Attempting to cancel an invoice with status SENT, PARTIAL, or PAID

On Approve

Invoice status → CANCELLED.

On Reject

Invoice remains in its original status (SENT / PARTIAL / PAID).

6. Manual Stock Adjustment

Trigger

Stock movement type is MANUAL_ADJUSTMENT or DAMAGE

On Approve

material.currentStock is updated atomically by the adjustment quantity.

On Reject

Stock movement marked REJECTED. currentStock unchanged.

Appendix CGlossary of Terms

TermDefinition
BOMBill of Materials — the recipe specifying every material, quantity formula, and cost for a curtain style.
DropThe finished length of a curtain from the top of the track to the hem, in centimetres.
Exchange RateThe number of Ghana Cedis (GHS) per 1 US Dollar (USD). Updated manually by an Admin when the rate changes.
Fullness RatioThe ratio of fabric width to track width. 2.0 = double fullness (the fabric is twice as wide as the finished curtain, creating gathers).
GRNGoods Receipt Note — a document recording what materials were physically received from a supplier, when, and at what cost.
Job CardA production work order for the workshop, listing the materials to be cut and sewn for a specific order.
Landed CostThe total cost of goods including purchase price, freight, import duties, and port clearing fees. The "true" cost by the time goods reach your warehouse.
LiningA secondary fabric stitched behind the main fabric for opacity, insulation, or structure.
Markup RatioThe percentage added above total cost to arrive at a selling price. E.g. a 0.35 markup on a GHS 100 cost gives a GHS 135 selling price.
OverheadIndirect business costs (electricity, rent, maintenance) that cannot be assigned to a single job but must be recovered across all jobs.
Overhead RateGHS per labour hour charged to each job to recover factory overhead. Calculated as total factory overhead ÷ total productive hours.
POPurchase Order — a formal document sent to a supplier to request goods at agreed prices.
WACWeighted Average Cost — the method used to recalculate material unit costs each time new stock is received, blending existing and new stock costs proportionally.
VATValue Added Tax — a consumption tax applied to sales. The rate is configurable in Settings.

Appendix DQuick Reference — Common Workflows

New Customer Job (Full Cycle)

  1. Customers → New Customer → enter details and window measurements
  2. Quotes → New Quote → select customer, add line items per window, set fabric, apply discount if needed → Create Quote
  3. Email or print the quote → customer confirms → Mark as Accepted
  4. Convert to Order → enter deposit → Confirm
  5. Production → job card appears → workshop issues materials and updates status
  6. Order delivered → Invoices → New Invoice → select order → Create → Record full payment

Restocking Materials

  1. Dashboard alerts or Inventory page shows low stock
  2. Purchasing → New PO → select supplier, add materials, set quantities and costs → Create PO
  3. Open PO → Submit for Approval → wait for approval
  4. Once approved → Send to Supplier (or download PDF)
  5. When goods arrive → open PO → Record Goods Receipt → enter received quantities
  6. If freight/clearing invoice arrives later → Landed Costs → New Entry → allocate to GRN → Post

End-of-Month Overhead Review

  1. Expenses → log all expenses for the month with correct Type (Admin/Factory/Shared)
  2. Reports → Overhead → set date range to the month → Generate
  3. Review the Suggested Overhead Rate
  4. If it differs significantly from the current Settings rate → Settings → General → update Overhead Rate → Save

Stocktake Correction

  1. Count physical stock
  2. Inventory → open the material → Adjust Stock
  3. Select MANUAL_ADJUSTMENT → enter the correct quantity difference → add a note explaining the count
  4. Submit → approval request sent to Approvals inbox
  5. Admin/Accounts approves → stock level updates
